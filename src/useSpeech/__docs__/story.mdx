A hook for speech synthesis using browser API

## Reference

```ts
type SpeechOptions = {
	voice?: SpeechSynthesisVoice;
	rate?: number;
	pitch?: number;
	volume?: number;
	lang?: string;
};

type SpeechPlayStatus = 'init' | 'play' | 'pause' | 'end' | 'error';

export type SpeechState = {
	isPlaying: boolean;
	status: SpeechPlayStatus;
	errorMessage?: string;
};
```

#### Arguments

- **text** _`string`_ - Text to be spoken

- **state** _`SpeechState`_ - Represents the current state of speech synthesis.
